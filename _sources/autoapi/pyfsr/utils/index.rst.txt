pyfsr.utils
===========

.. py:module:: pyfsr.utils

.. autoapi-nested-parse::

   Internal helpers shared across the SDK.

   Small, dependency-light utilities: file upload/download
   (:mod:`~pyfsr.utils.file_operations`), IRI parsing and construction
   (:mod:`~pyfsr.utils.iri`), and input validation
   (:mod:`~pyfsr.utils.validation`). These support the higher-level client and API
   layers and are rarely imported directly.

.. py:class:: APIKeyAuth(base_url: str, api_key: str, verify_ssl: bool = True)

   Bases: :py:obj:`pyfsr.auth.base.BaseAuth`


   API Key authentication handler for FortiSOAR.

   API key authentication has several limitations:
   - Cannot use /auth endpoints
   - Cannot export configurations

   :param base_url: Base URL of the FortiSOAR instance
   :param api_key: The FortiSOAR API key
   :param verify_ssl: Whether to verify SSL certificates. Defaults to True.

   :raises APIError: If API key validation fails

   .. rubric:: Example

   >>> auth = APIKeyAuth(
   ...     base_url="https://fortisoar.example.com",
   ...     api_key="your-api-key"
   ... )
   >>> headers = auth.get_auth_headers()

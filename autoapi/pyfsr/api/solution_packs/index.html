find_installed_pack(search_term: str) → Dict[str, Any] | None[source]#

Find a single installed solution pack by name, label, or description. Returns only the first matching pack found. For multiple results, use search_installed_packs() instead.

Parameters:

search_term – Name, label, or description to search for

Returns:

The first matching solution pack object, or None if no matches

Return type:

Dict[str, Any]

Example

# Find single installed pack
pack = client.solution_packs.find_installed_pack("SOAR Framework")
if pack:
    print(f"Found pack: {pack['name']}")
search_installed_packs(search_term: str = '', limit: int = 30) → List[Dict[str, Any]][source]#

Search for all installed solution packs matching the search criteria.

Parameters:

  • search_term – Name, label, or description to search for

  • limit – Maximum number of results to return (default 30)

Returns:

List of matching solution pack objects

Return type:

List[Dict[str, Any]]

Example

# Search for multiple installed packs
packs = client.solution_packs.search_installed_packs(
    search_term="SOAR",
    limit=10
)
for pack in packs:
    print(f"Found pack: {pack['name']}")

export_pack(pack_identifier: str, output_path: str | None = None, poll_interval: int = 5) → str[source]#

Export a solution pack by name, label, or search term.

Parameters:

  • pack_identifier – Name, label or search term to find the pack

  • output_path – Optional path to save exported file

  • poll_interval – How often to check export status in seconds

Returns:

Path where the exported file was saved

Example

# Export a solution pack by name
